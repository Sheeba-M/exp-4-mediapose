# MediaPipe Pose Estimation

📌 Overview
This project demonstrates real-time human pose estimation using MediaPipe and OpenCV. It detects body landmarks such as hands, legs, and head from a webcam or video.

 🎯 Objective
- To detect human body keypoints
- To track movements in real-time
- To visualize pose landmarks

 ⚙️ Technologies Used
- Python
- OpenCV
- MediaPipe

 📷 Features
- Real-time pose detection
- Works with webcam and video
- Displays body skeleton and landmarks
- 
 🧪 How It Works
1. Capture video using OpenCV
2. Process frames using MediaPipe Pose
3. Detect body landmarks
4. Draw landmarks on screen
5. Display output
